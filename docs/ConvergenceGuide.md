# 收敛调优与诊断指南

## 收敛判定
- 残差为平均穿透范数，需低于收敛阈值。
- 主循环达到阈值或迭代上限停止；单步迭代在变化量很小或达到上限时停止。

## 常用参数
- 主循环最大迭代：500（建议范围 360–600）
- 单步最大迭代：15（建议范围 12–20）
- 收敛阈值：0.0012（建议范围 0.001–0.0015）
- 松弛因子 ACCELERATION：1.2–1.5，最大不超过 1.8
- 容器调整系数：0.04–0.06，早期可到 0.08

## 调整策略
- 告警出现时，优先：主迭代 +20–50%、步迭代 +20–50%、ACCELERATION 下调。
- 需要更高精度时，适当降低阈值，同时增加迭代并下调 ACCELERATION。
- 早期残差很高时，略增容器调整系数以快速脱离高压重叠。

## 诊断方法
- 查看计算页收敛面板：曲线应整体下降并靠近阈值线。
- 批量模拟成功率≥95%为健康状态；主循环迭代数稳定。
- 使用小规模扫参，选择“成功率高且迭代数适中”的组合。

## 注意事项
- 阈值越小越难收敛，需要更多迭代与更谨慎的步长。
- 过大的 ACCELERATION 会导致曲线反弹或震荡。
